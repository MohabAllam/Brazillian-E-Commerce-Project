# Epsilon Ai Data Science Diploma (Final Poject)
# Introduction
### The dataset is about Olist a Brazilian ecommerce store which has information of 100k orders from 2016 to 2018 made at multiple marketplaces in Brazil where its features allow viewing an order from multiple dimensions: from order status, price, payment and freight performance to customer location, product attributes and finally reviews written by customers.
# Project Objectives
### In this project we are mainly helping Sales & Marketing team in many directions such as :
* Gain useful insights to take Data-Driven decisions.
* Customer Sgmentation.
* Customer Satisfaction Prediction.
* Future Sales  Forecasting.
